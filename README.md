The Expense Tracker is a simple and user-friendly web application developed using HTML, CSS, and JavaScript. This project helps users record, manage, and track their daily expenses efficiently.
Users can add expenses by entering the expense title, amount, and category such as Food, Transport, Shopping, Bills, or Others. The application automatically calculates and displays the total expenses in real time. Each expense entry can also be deleted when no longer needed.
